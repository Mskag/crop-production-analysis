# crop-production-analysis
This project analyzes crop production trends in India using agricultural datasets.  
## Project Description

This project performs **crop production analysis for India** using agricultural data sourced from the **UPAg Government dataset**.  
It processes and analyzes the data to generate meaningful insights through visualizations such as top crops, top producing states, production trends, and area–yield relationships.

---

## Techniques & Packages Used

- **Data Handling:**
  - `pandas` → data loading, filtering, grouping, and transformation  

- **Data Visualization:**
  - `matplotlib` → bar charts, line plots, and trend analysis  
  - `seaborn` → improved visualization styling  

- **Data Processing Techniques:**
  - Filtering data by **year and season**
  - Handling missing and inconsistent values  
  - Aggregation using **groupby and pivot tables**
  - Selecting the **latest estimation cycle** for accurate results  

---

## Workflow / Approach

### 1. Data Collection
- Dataset obtained from UPAg government agricultural records  

### 2. Data Preprocessing
- Cleaning numeric columns  
- Filtering by selected year and season  
- Selecting the most complete estimation cycle  

### 3. Exploratory Data Analysis (EDA)
- Identifying top crops and top states by production  
- Analyzing production trends over time  
- Studying relationships between area, yield, and production  

### 4. Visualization
- Bar charts for ranking crops and states  
- Line plots for trend analysis  
- Scatter/bubble plots for multi-variable relationships
